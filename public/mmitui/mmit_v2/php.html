Objective

PHP Web Developer Bootcamp @Yangon (တစ်နေ့ ၇နာရီ - ၆ပတ် ) သင်တန်း ဖြစ်ပါတယ်။ Project (၅)ခု ပြီးစီးအောင် သင်တန်းကာလတွင် ပြုလုပ် ပြီး ကိုယ်ပိုင် ဝက်ဆိုက်ပေါ် သို့ ကိုယ်တိုင်တင်ရမည် ဖြစ်သည့်အတွက် Junior Developer အဖြစ် အလုပ်ဝင်ရန် အဆင်သင့်ဖြစ်အောင် ပြင်ဆင်ပေးခြင်းဖြစ်ပါသည်။ Project ပြီးစီးအောင် ပြုလုပ်နိုင်မှ သာ Certificate ပေးမည်ဖြစ်ပါသည်။

What you'll learn

  • Make a REAL web applications
  • Create static HTML and CSS portfolio sites.
  • Use Bootstrap to create good-looking responsive layouts.
  • Make a beautiful, responsive portfolio page
  • Think like a developer.
  • Write web apps with full authentication
  • Continue to learn and grow as a developer long after the course ends.

Course Outline

  1. Web Design ( HTML5, CSS3,JavaScript, JQuery, Bootstrap4) and Mini Project
  2. Database Technology ( MySQL )
  3. Vue JS and PHP MVC MySQL RESTFul API (CI / Laravel Framework) and Mini Project
  4. PHP CMS ( Wordpress ) and Mini Project
  5. Software Engineering Fundamental
  6. Group Project Presentation

Facility

  • နယ်ကျောင်းသူများအတွက် Aircon/Internet အပါ အဆောင်စီစဉ်ပေးမည်
  • Certificate ပေးမယ့်အပြင် သင်တန်းသားများ အားလုံး ကိုယ်ပိုင် Domain / Hosting တစ်ခုစီ တစ်နှစ်စာ အခမဲ့ ရရှိမည်ဖြစ်ပါသည်
  • သင်တန်းပြီးဆုံးသည်နှင့် ရန်ကုန်မြို့ရှိ Web Development Company များတွင် Junior Web Developer အဖြစ် အလုပ်ဝင်နိုင်မည့် အခွင့်အရေး ရရှိမည်ဖြစ်ပါသည်
