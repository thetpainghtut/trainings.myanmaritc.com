Objective

Mobile Development ပိုင်းကို စိတ်ဝင်စားပြီး Android Application ကို ကိုယ်ပိုင် Idea နဲ့ ကိုယ်တိုင်ဖန်တီးရေးသားချင်သူများ | ပြည်တွင်းပြည်ပအိုင်တီကုမ္ပဏီ များတွင် Android Developer အဖြစ် ဝင်ရောက်လုပ်ကိုင်လိုသူများ | Programming OOP Concept အခြေခံသိရှိနားလည်ထားသူတိုင်းတက်ရောက်နိုင်ပါတယ်။ Android Developer အဖြစ် အလုပ်စဝင်ဖို့ဆိုရင် မဖြစ်မနေ Java ကို သိရှိဖို့လိုပါတယ်။ Java မရလို့ Android Developer လုပ်လို့မရဘူးဆိုတာ မဟုတ်ပါဘူးနော် ... အခုနောက်ပိုင်း ခေတ်စားလာတဲ့ Kotlin တို့ Flutter တို့က Java ကို အခြေခံလောက်သိထားပြီး Android ပိုင်း စိတ်ဝင်စားတဲ့သူတိုင်း လေ့လာနိုင်ပါတယ်။ Android Application တစ်ခု ရေးနိုင်ဖို့အတွက် Low Level မှ High Level အထိ ရောက်အောင် သင်ကြားပေးမှာဖြစ်လို့ Programming OOP Concept အခြေခံသိရှိနားလည်ထားသူတိုင်း တက်ရောက်နိုင်ပါတယ်။

What you'll learn

  • Make a REAL android applications
  • Create custom application with api.
  • Use Flutter or Kotlin to create good-looking android app.
  • Make a beautiful, application
  • Think like a developer.
  • Write android apps with full authentication
  • Continue to learn and grow as a developer long after the course ends.

Course Outline

  1. Introduction to Java for Android
  2. Introduction to Kotlin Programming language
  3. UI Layout Design
  4. Android Lifecycle
  5. Activity and Fragment
  6. App Navigation
  7. App Architecture
  8. Communicate with API service
  9. Build App with Firebase
  10. Android Jetpack
  11. Material Design
  12. Using third party libraries
  13. Publish the application to Play Store
  14. Introduction to Flutter
  15. Building Layouts
  16. Navigation and Routing
  17. State Management
